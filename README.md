## 🚀 Overview

**Vyapar Saathi** is a lightweight, mobile-first web application designed to help small shop owners manage their inventory and sales efficiently without technical complexity.

It combines **barcode scanning, real-time tracking, and smart insights** into a single easy-to-use interface.

---

## 🌟 Key Features

### 📦 Inventory Management
- Add products manually or via barcode scanning  
- Auto-fetch product details using API  
- Edit and delete inventory items  

### 🛒 Smart Selling (Cart Mode)
- Scan items directly into cart  
- Real-time quantity updates  
- Fast checkout flow  

### 📊 Dashboard Analytics
- Total products  
- Total items in stock  
- Revenue tracking  

### ⚠️ Insights & Alerts
- Low stock detection  
- Expiry alerts  
- Basic demand prediction (demo)  
- Smart inventory suggestions  

### 🔐 Authentication
- Firebase Email/Password login  
- Secure session handling  
- Protected dashboard access  

### 🌐 Multi-language Support
- English ✅  
- Hindi ✅  
- More languages (Coming Soon)  

### 📸 Barcode Scanner
- Camera-based scanning (mobile optimized)  
- Continuous scan mode for fast billing  
- Works best on HTTPS (Netlify)  

### 📱 Mobile-First Design
- Fully responsive UI  
- Smooth touch interactions  
- Designed for real shop environments
